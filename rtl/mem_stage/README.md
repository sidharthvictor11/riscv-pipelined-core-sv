# Memory Access Stage (MEM)

## Status

🚧 Under Development

The Memory Access (MEM) stage will be responsible for interfacing with the data memory and executing load/store instructions in the RV32I ISA.

## Planned Features

- Data memory interface
- Load instructions (LB, LH, LW, LBU, LHU)
- Store instructions (SB, SH, SW)
- Byte, half-word, and word memory accesses
- Memory read/write control logic

## Future Modules

- Data Memory
- Memory Access Controller
- Load Data Selection
- Store Data Path

> This stage will be implemented after the Execute (EX) stage is completed.
